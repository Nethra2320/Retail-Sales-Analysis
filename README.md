# Retail-Sales-Analysis

Retail Sales Analysis Project using SQL, Excel, and Power BI. This project includes data cleaning, KPI creation, interactive dashboards, and business insights to support data-driven decision-making.

# Project Overview
This project is an end-to-end Retail Sales Analysis developed using Excel, SQL, Python, and Power BI. The objective is to analyze retail sales data, identify business trends, and create an interactive dashboard that provides meaningful insights for decision-making. The project demonstrates the complete data analytics workflow, including data cleaning, data analysis, visualization, and dashboard development.

# Project Objectives
-Analyze retail sales performance. -Identify sales trends across different years. -Compare sales across product categories and cities. -Understand customer purchasing behavior. -Analyze customer payment preferences. -Build an interactive Power BI dashboard for business insights.

# Key Performance Indicators
-Total Sales: $447.31M -Total Orders: 37K -Average Order Value (AOV): 12K -Profit Margin: 17.40%

# Dashboard Features
-Interactive filters for Year, Category, and Region. -Sales analysis by city. -Customer type analysis (New vs Returning). -Payment method analysis. -Sales trend by year and category. -Revenue analysis by city and product subcategory. -KPI cards for business performance.

# Business Insights
-Achieved total sales of $447.31M. -Processed approximately 37K customer orders. -Maintained an average order value of 12K. -Achieved a profit margin of 17.40%. -Compared sales performance across cities and product categories. -Analyzed customer preferences based on payment methods and customer type.

#Conclusion
This project demonstrates practical skills in data cleaning, Excel, Power BI, data visualization, and business intelligence. It provides an interactive dashboard that helps businesses monitor sales performance and make data-driven decisions.
